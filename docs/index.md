
# Basics of plane-wave density functional theory

🛠️ **UNDER CONSTRUCTION** 🛠️

This guide is intended to be a crash course in density functional theory (DFT) and its practical implementation in plane-wave basis set codes used (primarily) to study crystalline materials.

Given my background and the focus of the Materials Project, the practical part of this guide will be aimed at VASP users.
A lot of the same concepts will hold for users of other plane-wave basis set codes such as Quantum Espresso, or augmented plane-wave codes like CP2K.
Users of Gaussian basis set codes (ex., alphabetically: Gaussian, ORCA, QCHEM, TURBMOLE) might be able to take away some high level concepts mostly about DFT.

This guide will also try to give you some background in high-throughput calculations.
